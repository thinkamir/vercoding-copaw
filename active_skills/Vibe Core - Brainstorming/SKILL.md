# Vibe Core: Brainstorming

来源：Vibe-Skills `core/skills/brainstorming`

## 适用场景
- 新功能构思
- 设计变更
- 实现前方案探索
- 创意型任务

## 核心规则
- **设计未获批准前，不启动实现**
- 先做理解与 trade-off 探索，再创建计划
- 获批后的下一步应当是**implementation planning**，而不是直接编码
