# Vibe Theme: Autonomous Builder

来源：Vibe-Skills `bundled/skills/autonomous-builder`

## 触发场景
- 用户要从需求持续推进到实现
- 项目需要连续迭代、逐步交付
- 需要在一个较长链路里自动推进：设计 -> 开发 -> 测试 -> 修复

## 推荐工作流
1. 先明确需求或读取现有计划
2. 建立当前状态锚点（文件、任务清单、Git 提交）
3. 以小步循环推进：PLAN -> BUILD -> TEST -> DEBUG
4. 每完成一个阶段都留下可验证结果
5. 对高风险动作单独确认，不越权外扩

## 协同建议
- 任务总控：配合 `Vibe Skills`
- 先产计划：配合 `Vibe Core - Writing Plans`
- 严格开发闭环：配合 `Vibe Core - TDD Guide`
- 需要拆任务并行：配合 `Vibe Core - Subagent Development`

## 不适合
- 用户只要一次性小修小补
- 涉及生产部署、基础设施变更、工作区外操作但未获批准
